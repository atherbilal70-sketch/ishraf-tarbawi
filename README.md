# دليل الإشراف التربوي

تطبيق ويب تقدمي (PWA) مرجعي شخصي، مصمم ليجمع أهم القوانين والتعليمات والأدوات المساعدة الخاصة بمنصب معاون مدير عام الإشراف التربوي في وزارة التربية العراقية.

## المزايا الحالية

- تصفح القوانين والتعليمات حسب التصنيف (8 تصنيفات جاهزة، فارغة حالياً بانتظار المحتوى)
- بحث فوري في كل المحتوى
- مفضلة شخصية + ملاحظات على كل قانون
- نماذج مخاطبات رسمية جاهزة للنسخ (كتاب عام، إحالة، طلب إجازة، تقرير زيارة)
- جهات اتصال سريعة ومواعيد/تذكيرات (تُحفظ محلياً على الجهاز)
- وضع ليلي وحجم خط قابل للتعديل
- يعمل بدون إنترنت بعد أول فتح (Service Worker) ويُضاف للشاشة الرئيسية على آيفون

## طريقة التثبيت على آيفون

1. افتح رابط الموقع عبر متصفح **Safari**.
2. اضغط زر المشاركة (المربع مع السهم للأعلى).
3. اختر **"إضافة إلى الشاشة الرئيسية"**.
4. سيظهر التطبيق بأيقونته الخاصة ويعمل بملء الشاشة بدون شريط المتصفح.

## كيفية إضافة القوانين والتعليمات

كل المحتوى القانوني يُدار من ملف واحد: `data/laws.json`.
كل تصنيف (`category`) يحتوي مصفوفة `laws`، وكل قانون بهذا الشكل:

```json
{
  "id": "unique-id-فريد",
  "title": "عنوان القانون أو التعليمات",
  "summary": "ملخص قصير (اختياري)",
  "body": "النص الكامل هنا...",
  "source": "المصدر (رقم الأمر، التاريخ، الجهة المصدرة)",
  "tags": ["كلمة مفتاحية 1", "كلمة مفتاحية 2"]
}
```

أرسل النصوص أو الملفات الرسمية (PDF/Word) وسيتم تحويلها وإضافتها هنا بدقة، دون أي اجتهاد أو تخمين في المحتوى القانوني.

## ملاحظة مهمة

هذا التطبيق أداة مساعدة للتنظيم والوصول السريع فقط، وليس بديلاً عن الرجوع للنصوص الرسمية والجريدة الرسمية عند اتخاذ أي قرار إداري مهم.
