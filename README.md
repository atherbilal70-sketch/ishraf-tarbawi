# دليل الإشراف التربوي

تطبيق ويب تقدمي (PWA) مرجعي شخصي، مصمم ليجمع أهم القوانين والتعليمات والأدوات المساعدة الخاصة بمنصب معاون مدير عام الإشراف التربوي في وزارة التربية العراقية.

## المزايا الحالية

- تصفح القوانين والتعليمات حسب التصنيف (8 تصنيفات جاهزة، فارغة حالياً بانتظار المحتوى)
- بحث فوري في كل المحتوى
- **مساعد ذكاء اصطناعي** يجيب على أسئلة حرة استناداً للقوانين المضافة فقط (يحتاج إعداد لمرة واحدة — انظر القسم أدناه)
- مفضلة شخصية + ملاحظات على كل قانون
- نماذج مخاطبات رسمية جاهزة للنسخ (كتاب عام، إحالة، طلب إجازة، تقرير زيارة)
- جهات اتصال سريعة ومواعيد/تذكيرات (تُحفظ محلياً على الجهاز)
- وضع ليلي وحجم خط قابل للتعديل
- يعمل بدون إنترنت بعد أول فتح (Service Worker) ويُضاف للشاشة الرئيسية على آيفون

## طريقة التثبيت على آيفون

1. افتح رابط الموقع عبر متصفح **Safari**.
2. اضغط زر المشاركة (المربع مع السهم للأعلى).
3. اختر **"إضافة إلى الشاشة الرئيسية"**.
4. سيظهر التطبيق بأيقونته الخاصة ويعمل بملء الشاشة بدون شريط المتصفح.

## المساعد الذكي (اسأل عن مشكلتك)

ميزة جديدة تتيح كتابة سؤال بأسلوب طبيعي (مثل: "المعلم تغيب 5 أيام بدون عذر شنو الإجراء؟")
والحصول على إجابة مبنية حصراً على القوانين المضافة داخل التطبيق (بدون اختراع أي معلومة).

هذه الميزة تحتاج إعداداً لمرة واحدة من طرفك، لأنها تستدعي خدمة ذكاء اصطناعي خارجية (Claude من Anthropic)
عبر خادم وسيط صغير يحفظ المفتاح بسرية. **التطبيق نفسه يعمل بالكامل بدون هذا الإعداد** — فقط زر
"اسأل الذكاء الاصطناعي" لن يعمل حتى تُكمل الخطوات التالية.

### الخطوة 1 — إنشاء مفتاح Anthropic API

1. افتح https://console.anthropic.com وأنشئ حساباً.
2. من القائمة الجانبية اختر **API Keys** ثم **Create Key**، وسمِّه مثلاً `ishraf-tarbawi`.
3. انسخ المفتاح فوراً (يبدأ بـ `sk-ant-...`) — يظهر مرة واحدة فقط ولا يمكن رؤيته لاحقاً.
4. من **Plans & Billing** أضف وسيلة دفع لتفعيل الاستخدام. تكلفة كل سؤال شخصي زهيدة جداً (كسور من السنت عادةً).
5. **لا ترسل هذا المفتاح لي أو لأي جهة عبر الدردشة أو أي وسيلة أخرى** — أدخله مباشرة في Cloudflare كما في الخطوة 3.

### الخطوة 2 — نشر الموقع نفسه عبر Cloudflare Pages

المستودع حالياً فقط ملفات على GitHub وغير منشور كموقع فعلي. الأسهل هو ربطه بـ Cloudflare Pages (مجاني):

1. أنشئ حساباً مجانياً على https://dash.cloudflare.com
2. **Workers & Pages** ← **Create** ← تبويب **Pages** ← **Connect to Git**
3. اربط حساب GitHub وامنح الوصول لمستودع `ishraf-tarbawi`
4. في إعدادات البناء اتركها كما هي (بدون build command، مجلد الإخراج هو الجذر `/`) ثم **Save and Deploy**
5. ستحصل على رابط مثل: `https://ishraf-tarbawi.pages.dev` — أي تحديث تدفعه للمستودع لاحقاً يُنشر عليه تلقائياً خلال دقيقة تقريباً

> ⚠️ بما أن الموقع سيصبح على رابط عام، أي شخص يعرف الرابط يقدر يفتحه (حتى مع أن مستودع GitHub خاص). إذا كنت تريد قصر الوصول على أشخاص محددين (مثل والدك فقط)، أخبرني بعد النشر وأضيف حماية مجانية عبر **Cloudflare Access** (تسجيل دخول برمز يُرسل للبريد الإلكتروني، بدون كلمة مرور يحفظها أحد).

### الخطوة 3 — نشر الخادم الوسيط (Worker) الخاص بالذكاء الاصطناعي

1. من نفس حساب Cloudflare: **Workers & Pages** ← **Create** ← **Create Worker**
2. سمِّه مثلاً `ishraf-tarbawi-ai` ← **Deploy** (سينشر نسخة افتراضية فارغة أولاً)
3. اضغط **Edit code**، احذف الكود الموجود، وألصق بدلاً منه محتوى الملف [`worker/worker.js`](worker/worker.js) من هذا المستودع بالكامل
4. **Save and Deploy**
5. اذهب لتبويب **Settings → Variables and Secrets** وأضف سرّين (Secret، وليس Text عادي):
   - `ANTHROPIC_API_KEY` = المفتاح الذي أنشأته في الخطوة 1
   - `APP_SHARED_KEY` = `c5a01526b28257b6884db2407922decc5a80968ca862bf84` (طابق هذه القيمة تماماً مع ما هو موجود في `js/config.js` — لا تغيّرها)
6. احفظ، وانسخ رابط الـ Worker الظاهر أعلى الصفحة (ينتهي بـ `workers.dev`)

### الخطوة 4 — الربط النهائي

أرسل لي رابط الـ Worker فقط (مثل `https://ishraf-tarbawi-ai.xxxx.workers.dev`) — هذا رابط عام وليس سرياً، لا مشكلة بمشاركته.
سأضعه في `js/config.js` وأدفع التحديث، وتعمل ميزة الذكاء الاصطناعي مباشرة بعدها.

## كيفية إضافة القوانين والتعليمات

كل المحتوى القانوني يُدار من ملف واحد: `data/laws.json`.
كل تصنيف (`category`) يحتوي مصفوفة `laws`، وكل قانون بهذا الشكل:

```json
{
  "id": "unique-id-فريد",
  "title": "عنوان القانون أو التعليمات",
  "summary": "ملخص قصير (اختياري)",
  "body": "النص الكامل هنا...",
  "source": "المصدر (رقم الأمر، التاريخ، الجهة المصدرة)",
  "tags": ["كلمة مفتاحية 1", "كلمة مفتاحية 2"]
}
```

أرسل النصوص أو الملفات الرسمية (PDF/Word) وسيتم تحويلها وإضافتها هنا بدقة، دون أي اجتهاد أو تخمين في المحتوى القانوني.

## ملاحظة مهمة

هذا التطبيق أداة مساعدة للتنظيم والوصول السريع فقط، وليس بديلاً عن الرجوع للنصوص الرسمية والجريدة الرسمية عند اتخاذ أي قرار إداري مهم.
